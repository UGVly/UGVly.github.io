# Welcome to My Notes Blog

This directory is the source for the blog section under `/blog/`.

- Put your daily markdown notes here.
- Run `npm.cmd run build:blog`.
- The script will generate static HTML pages in `blog/`.

## Tips

- You can organize notes in subfolders.
- The first `# Heading` will be used as the post title.
- This works well for research memos, reading notes, and experiment logs.
